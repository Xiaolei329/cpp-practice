# C++ 学习笔记与练习代码

这个仓库用于记录我在学习 C++ 过程中的代码练习和阶段性整理。  
内容会随着学习进度持续补充，尽量保持原始学习痕迹，方便回顾与复盘。

## 仓库内容

- C++ 基础语法练习
- 面向对象相关示例（构造、析构、拷贝构造、友元等）
- 引用、字符串与动态内存（new）练习
- 运算符重载示例
- 课程资料与习题实现代码
