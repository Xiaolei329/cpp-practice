# C++ 学习记录

这是我在学习 C++ 过程中的练习仓库。  
里面会持续上传学习时写的源码和对应的简要介绍，主要用于记录自己的学习过程，也方便他人（比如 HR）了解我的技术成长轨迹。

## 仓库内容

- C++ 基础语法练习
- 面向对象相关示例（构造、析构、拷贝构造、友元等）
- 引用、字符串与动态内存（new）练习
- 运算符重载示例
- 课件与习题实现代码

## 仓库名称和介绍怎么写（不刻意、真实学习）

### 1) 仓库名称建议（任选一个自然的）

- `cpp-learning-notes`
- `cpp-practice-notes`
- `cpp-study-record`

这些名字都比较直白，重点是“学习/练习/记录”，不会显得刻意包装。

### 2) 仓库简介（Description）可直接填写

可选中文：

> C++ 学习过程中的练习代码与笔记整理，持续更新。

可选中英结合：

> Personal C++ learning repository with practice code and notes, updated along the way.

这个仓库以“真实学习记录”为主，不追求项目包装感；后续会按学习进度继续补充内容。
