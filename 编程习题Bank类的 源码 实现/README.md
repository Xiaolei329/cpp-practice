银行账户管理系统 (Bank Account Management System)
一个基于C++的简单银行账户管理系统，支持账户创建、存款、取款和交易记录查询功能。

功能特性
核心功能
✅ 账户创建 - 初始化账户信息（账号、创建日期、初始余额）

✅ 存款操作 - 记录存款金额和日期

✅ 取款操作 - 余额检查并记录取款信息

✅ 交易查询 - 显示近期的交易明细记录

技术特点
使用面向对象编程设计

循环记录最近30条交易记录（模拟一个月）

自动余额检查和取款验证

结构化的交易记录存储

类结构说明
Bank 类
主要管理银行账户的核心功能：

成员变量
account - 账户号码

creatTime - 账户创建日期

money - 当前账户余额

dealRecord - 交易记录数组（最多30条）

index - 当前记录索引位置

成员方法
Bank(string, string, double) - 构造函数，创建新账户

SaveMoney(double, string) - 存款操作

GetMoney(double, string) - 取款操作

showDealRecord() - 显示交易记录

DealRecord 结构体
存储单条交易记录：

type - 交易类型（"存钱"/"取钱"）

date - 交易日期

dealMoney - 交易金额

使用方法
编译运行
bash
g++ -o bank_system main.cpp
./bank_system
基本操作示例
cpp
// 创建账户
Bank user1 = Bank("123123", "2023-3-27", 1000);

// 存款操作
user1.SaveMoney(200, "2023-3-27");

// 取款操作
user1.GetMoney(100, "2023-3-30");

// 查询交易记录
user1.showDealRecord();

技术细节
记录管理
使用循环数组存储交易记录

最多保存30条最近交易记录

自动覆盖最旧的记录当达到上限

错误处理
取款时自动检查余额是否充足

余额不足时会显示提示信息并取消操作
